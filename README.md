# Employee Attrition Prediction using Machine Learning

## Overview

This project predicts employee attrition using Machine Learning techniques on the IBM HR Analytics dataset.

The project covers:

- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Classification Models
- Model Evaluation
- Business Recommendations

---

## Machine Learning Models

- Logistic Regression
- Random Forest Classifier
- Gradient Boosting Classifier
